# Supply Chain & Logistics Power BI Dashboard
This project simulates a real-world analytics scenario using a custom dataset based on Samsung. It covers the entire supply chain process, from raw materials to final customer delivery, aimed at helping a Data Analyst make informed business decisions.
# Project Features & Modules
Power Query ETL: Data cleaning and transformation from a folder of CSV files.
Data Modeling: Establishing a robust star schema with one-to-many relationships between Dimension (Products, Customers) and Fact (Sales, Inventory) tables.
DAX Measures: Creation of advanced DAX formulas for key performance indicators.
Interactive Visualization: A dynamic dashboard featuring logistical maps, stock level charts, and slicers.
Key KPIs & Metrics
Total Revenue: Total sales generated.
Perfect Order Percentage: A custom metric calculating on-time delivery without defects.
Stock Turnover Rate: Analysis of inventory movement
